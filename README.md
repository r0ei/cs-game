# Catch The Shape
The game is basically a game that you have to try to click on a shape that appears randomly on the screen for a second, after a second it will be deleted from the screen and another one will appear randomly. Clicking on the shape adds you 1 score to the game statistics. There is an option to change the number of rounds of the game, you must select an integer that is small or equal to 20.
